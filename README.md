## Overview

ElectroShop is a complete e-commerce ecosystem consisting of:

* RESTful Backend API (.NET 8)
* Web Administration & Customer Portal (React + Vite)
* Native Android Mobile Application (Kotlin + Jetpack Compose)

The system supports product management, order processing, authentication, payment integration, customer support chat, inventory management, analytics, and mobile commerce.
